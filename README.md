# PAT-Project

Колекція невеликих веб-проєктів для планування часу та перегляду подій.

## Структура

```text
PAT-Project/
├── work_calendar/          # Календар змін
│   └── index.html
├── SundayPlan/            # Афіша / арт-програма
│   └── index.html
└── README.md
```

## Проєкти

### Work Calendar (`work_calendar/`)

Календар графіка роботи з адаптивним інтерфейсом і локальним збереженням налаштувань.

Основні можливості:

- готові графіки `5/5`, `4/4`, `5/2`, `3/3`, `2/2` і кастомний цикл
- вибір дати старту циклу для правильного перерахунку змін
- подвійний клік по дню для ручного перемикання статусу
- позначки державних свят у календарі та в модальному вікні дня
- поточна погода для Борисполя у шапці
- денний прогноз для Борисполя дрібним рядком під поточною погодою
- перемикання світлої/темної теми
- кастомізація кольорів робочих і вихідних днів
- swipe-навігація між місяцями на мобільних пристроях
- збереження графіка, кольорів, теми та ручних змін у `localStorage`

Технічні деталі:

- `Vanilla HTML/CSS/JavaScript`
- погодні дані через `Open-Meteo API`
- без зовнішніх залежностей і збірки

### SundayPlan (`SundayPlan/`)

Окремий статичний веб-проєкт для програми подій та розваг.

## Запуск

Оскільки обидва проєкти статичні, достатньо відкрити відповідний `index.html` у браузері.

Приклади:

```bash
# Work Calendar
start work_calendar/index.html

# SundayPlan
start SundayPlan/index.html
```

Альтернатива: відкрити файли напряму з провідника або через Live Server у редакторі.

## Work Calendar: коротко про використання

- натисни на день, щоб відкрити деталі
- зроби подвійний клік по дню, щоб змінити його статус
- використай кнопку з шестернею для кольорів
- використай кнопку графіка для зміни циклу та дати старту
- кнопка `Сьогодні` повертає до поточного місяця
- кнопка `↻` у погодному віджеті оновлює дані

## Примітки

- календар орієнтований на локальне використання в браузері
- для погоди потрібен доступ до `api.open-meteo.com`
- без мережі календар продовжує працювати, але погодний віджет не оновлюється
