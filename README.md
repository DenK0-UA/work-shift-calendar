# PAT-Project 📅

Колекція веб-додатків для планування та організації часу.

## Структура проекту

```
PAT-Project/
├── work_calendar/          # Графік роботи (5/5)
│   └── index.html         # Календар з погодою та розписом
├── SundayPlan/            # Арт-програма
│   └── index.html         # Програма подій в Києві
└── README.md
```

## Проекти

### 🗓️ Work Calendar (`work_calendar/`)

Сучасний календар графіка роботи з iOS-подібним дизайном.

**✨ Особливості:**

- 📱 Responsive дизайн (мобільний, планшет, десктоп)
- 🌓 Автоматична темна тема (слідування ОС)
- 🌤️ Інтеграція з метеоданими (Open-Meteo API)
- 📊 Flex 5/5 графік з легкою зміною днів
- 🎨 Кастомізація коліру робочих/вихідних днів
- 🎯 Подвійний клік на день для зміни статусу
- 📱 Swipe жести для навігації по місяцях (мобільні)
- ⭐ iOS 26 дизайн з глибокими тінями
- 💾 Збереження налаштувань в localStorage

**Логіка графіка:** Автоматичний розрахунок 5 днів роботи / 5 днів вихідних, з можливістю редагування окремих днів.

**Технологія:** Vanilla HTML, CSS (з медіа запитами), JavaScript

### 🎨 SundayPlan (`SundayPlan/`)

Елегантна програма подій та розваг для Києва.

**Особливості:**

- ✨ Преміум дизайн з gold і pink акцентами
- 🎭 Арт-програма для культурних закладів
- 🌆️ Інформація про события в місті
- 🎬 Сучасний веб-інтерфейс

**Технологія:** HTML, CSS, JavaScript

## 🚀 Використання

Кожен проект можна запустити просто відкривши `index.html` у браузері:

```bash
# Work Calendar
open work_calendar/index.html

# SundayPlan
open SundayPlan/index.html
```

## 🎨 Технічні деталі

**Work Calendar:**

- Vanilla JS (без залежностей)
- Responsive CSS з медіа запитами (< 480px, 768px+, 1024px+)
- LocalStorage для збереження налаштувань
- Open-Meteo API для погодних даних
- iOS-подібні ефекти та тіні

## 📝 Налаштування

- **Кастомізація коліру:** ⚙️ → вибір кольорів
- **Темна тема:** Автоматично слідує системним налаштуванням ОС
- **Зміна днів:** Подвійний клік = переключення статусу дня
- **Навігація:** Стрілки або swipe (мобільні)

Персональний проект для планування та організації часу.
