## 🗂️ Проєкт

### 🗓️ Work Calendar

Адаптивний календар графіка роботи з локальним збереженням налаштувань, кастомним циклом змін, погодою для обраної дати та кількома стилями інтерфейсу. Логіка стану теми, стилю і кольорів винесена в окремий модуль `settings-state.js`, модалка графіка винесена в `schedule-ui.js`, а статистика в `insights.js`.

**Можливості:**

- 🔁 Готові цикли: `5/5`, `4/4`, `5/2`, `3/3`, `2/2`
- ⚙️ Кастомний графік зі своїми днями роботи та вихідними
- 📅 Вибір дати старту циклу
- 👆 Явне редагування статусу дня в модальному вікні
- 📝 Нотатки для окремих дат
- 🎉 Позначки державних свят у календарі
- 🪟 Свято і прогноз погоди показуються в модальному вікні дня
- ↻ Оновлення прогнозу через `Open-Meteo` прямо в модалці дня
- 📊 Статистика за місяць, 3 місяці або рік у розкривному блоці
- 🎨 Налаштування кольорів робочих і вихідних днів
- 🧩 Пресети інтерфейсу: `Current`, `iOS 26`, `Material 3` з компактнішим виглядом на мобільних
- 🌗 Режими теми: `Авто`, `Світла`, `Темна`
- 📱 Swipe-навігація між місяцями на мобільних
- 💾 Збереження налаштувань у `localStorage`
- ↩️ Швидке скасування останньої ручної зміни дня

**Технології:**

- `HTML`
- `CSS`
- `Vanilla JavaScript`
- `Open-Meteo API`

## 🚀 Запуск

Проєкт статичний, тому його достатньо просто відкрити в браузері.

## 🧭 Як користуватись Work Calendar

- 🖱️ Натисніть на день, щоб побачити деталі
- 🔄 Відкрийте день і змініть його статус кнопками `Робочий / Вихідний / За графіком`
- 📝 У модальному вікні дня можна додати, зберегти або очистити нотатку
- ↩️ Після ручної зміни дня можна швидко натиснути `Скасувати`
- ⚙️ Відкрийте налаштування, щоб змінити стиль, тему і кольори
- 📆 Відкрийте меню графіка, щоб змінити цикл і дату старту
- 📊 Блок статистики можна розгорнути за потреби й перемикати період: `Місяць`, `3 міс`, `Рік`
- 📍 Кнопка `Сьогодні` повертає до поточного місяця
- ↻ У модалці дня можна окремо оновити прогноз для доступних дат

## 📝 Примітки

- 🌐 Для оновлення погоди потрібен доступ до `api.open-meteo.com`
- 🔌 Без інтернету календар працює далі, але прогноз погоди не оновлюється
- 🧩 Проєкт не потребує збірки чи сторонніх залежностей
