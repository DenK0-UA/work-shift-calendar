# 📚 PAT-Project

Невелика колекція веб-проєктів для планування часу, змін і перегляду подій.

## ✨ Що є в репозиторії

```text
PAT-Project/
├── work_calendar/      # Календар змін
│   └── index.html
├── SundayPlan/         # Афіша / арт-програма
│   └── index.html
└── README.md
```

## 🗂️ Проєкти

### 🗓️ Work Calendar

Адаптивний календар графіка роботи з локальним збереженням налаштувань і погодним віджетом.

**Можливості:**

- 🔁 Готові цикли: `5/5`, `4/4`, `5/2`, `3/3`, `2/2`
- ⚙️ Кастомний графік зі своїми днями роботи та вихідними
- 📅 Вибір дати старту циклу
- 👆 Подвійний клік по дню для ручної зміни статусу
- 🎉 Позначки державних свят у календарі
- 🪟 Свято показується і в модальному вікні дня
- 🌤️ Поточна погода для Борисполя у шапці
- 🌡️ Денний прогноз для Борисполя дрібним рядком під поточною погодою
- 🎨 Налаштування кольорів робочих і вихідних днів
- 🌗 Світла і темна тема
- 📱 Swipe-навігація між місяцями на мобільних
- 💾 Збереження налаштувань у `localStorage`

**Технології:**

- `HTML`
- `CSS`
- `Vanilla JavaScript`
- `Open-Meteo API`

### 🎭 SundayPlan

Статичний веб-проєкт для програми подій, афіші та міського дозвілля.

## 🚀 Запуск

Обидва проєкти статичні, тому їх достатньо просто відкрити в браузері.

```bash
# Work Calendar
start work_calendar/index.html

# SundayPlan
start SundayPlan/index.html
```

Альтернатива:

- відкрити файл напряму з провідника
- запустити через Live Server у редакторі

## 🧭 Як користуватись Work Calendar

- 🖱️ Натисни на день, щоб побачити деталі
- 🔄 Зроби подвійний клік по дню, щоб змінити його статус
- ⚙️ Відкрий налаштування, щоб змінити кольори
- 📆 Відкрий меню графіка, щоб змінити цикл і дату старту
- 📍 Кнопка `Сьогодні` повертає до поточного місяця
- ↻ Кнопка в погодному віджеті оновлює дані

## 📝 Примітки

- 🌐 Для оновлення погоди потрібен доступ до `api.open-meteo.com`
- 🔌 Без інтернету календар працює далі, але погодний віджет не оновлюється
- 🧩 Проєкт не потребує збірки чи сторонніх залежностей
