# PAT-Project 📅

Колекція веб-додатків для планування та організації часу.

## Структура проекту

```
PAT-Project/
├── work_calendar/          # Графік роботи (5/5)
│   └── index.html         # Календар з погодою та розписом
├── SundayPlan/            # Арт-програма
│   └── index.html         # Програма подій в Києві
└── README.md
```

## Проекти

### 🗓️ Work Calendar (`work_calendar/`)

Сучасний календар графіка роботи з адаптивним дизайном.

**Особливості:**

- 📱 Адаптивний дизайн (мобільний та десктоп)
- 🌓 Темна та світла тема
- 🌤️ Інтеграція з погодою
- 📊 Розпис робочих днів (Fluid 5/5)
- ⚡ Apple-like UI дизайн

**Технологія:** HTML, CSS, JavaScript

### 🎨 SundayPlan (`SundayPlan/`)

Елегантна програма подій та розваг для Києва.

**Особливості:**

- ✨ Преміум дизайн з gold і pink акцентами
- 🎭 Арт-програма для культурних закладів
- 🌆️ Інформація про события в місті
- 🎬 Сучасний веб-інтерфейс

**Технологія:** HTML, CSS, JavaScript

## 🚀 Використання

Кожен проект можна запустити просто відкривши `index.html` у браузері:

```bash
# Work Calendar
open work_calendar/index.html

# SundayPlan
open SundayPlan/index.html
```

## 📝 Примітки

- `.vscode/` папка містить локальні налаштування VS Code
- Проекти використовують тільки vanilla JavaScript (без фреймворків)
- Оптимізовані для мобільних пристроїв
